Description

Telegram bot is used for tagging media files for quick sending it in a chat. Bot tags your photos and stickers. Just send photo or media to the bot and he will ask you about tags for these medias. After saving you can get your pictures by tags in any chat. Just input
@taggsBot tag othertag ...
and bot will show you your pictures.

Start bot

Start bot (find @taggsBot between users or click by @taggsBot)

Create tags

Send photo or sticker to @taggsBot and input your tags.
Don't forget do it in chat with @taggsBot
You can send photo or sticker by means of other bot
You can forward it from other chat.
Or you can directly upload your media

Send tags

Choose saved media: just start input needed tag
@taggsBot tag othertag ...
and all medias with this tags will be available.
Of course you can input more than one tag.
If you want to see all your medias just input
@taggsBot %

Edit tags

If you have to edit your tags - select (or send repeatedly) your media and send it back to bot. Bot will ask you about new tags. Input and save.
For removing your tags from bot memory - send him needed media and press button "/delete".
Command
@taggsBot /delete
also will work.
